# Kegs

This Angular.js application displays keg information for a bar. Users may add, edit, and delete kegs and filter based on alcohol content and keg type.

## Installation

* `git clone <repository-url>`
* `cd kegs`
* `npm install`
* `bower install`
* `gulp serve`
* Visit [http://localhost:4200]
